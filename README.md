# Log Sentinel

A static log analysis and anomaly detection tool that scans uploaded logs, scores and tags critical anomalies with MITRE ATT&CK mappings, and automatically relays batched email alerts via EmailJS — with deduplication and rate limiting built in.

## Stack
- Vite + React + TypeScript
- EmailJS for alert relay

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`

## Deploy

Hosted on Vercel.
